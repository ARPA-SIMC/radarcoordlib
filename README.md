# Radarcoordlib

## Introduction

Radarcoordlib is a library to manage radar geodetic calculations.

## Documentation

The documentation available is generated by Doxygen in the doc/html 
directory.

## Software dependencies

Radarcoordlib needs [Geographiclib](http://geographiclib.sourceforge.net/).
Radarcoordlib should build and run on any reasonably recent Unix system, 
and has been tested on Debian based and Red Hat based distributions.
